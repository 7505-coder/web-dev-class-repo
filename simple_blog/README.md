# Simple Blog

A Flask-based simple blog web application for Experiment 5.

## Features
- Create blog posts
- Read and display all posts on the homepage
- Update existing posts
- Delete posts
- Basic styling and navigation

## Project Structure
- app.py
- requirements.txt
- templates/
- static/

## Run Locally
1. Install dependencies:
   - pip install -r requirements.txt
2. Start the app:
   - python app.py
3. Open the browser at:
   - http://127.0.0.1:5000/
